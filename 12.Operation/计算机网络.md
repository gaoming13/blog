### OSI七层模型 与 TCP/IP模型

```
7.应用层
  - 协议头:HTTP/FTP/SMTP/TELNET/Websocket
6.表示层
  - 协议头:加密解密,压缩解压缩,转换翻译
5.会话层
  - 协议头:端口号
4.传输层
  - 协议头:TCP/UDP
  - 接受上一层的数据,对数据进行分割,加上协议头,丢给网络层,保证数据有效到达对端
  - Socket: 提供对TCP/IP协议的封装,本身并不是协议,而是一个调用接口
    - 通过使用Socket编好的程序在应用层,如 http
3.网络层
  - 协议头:源IP+目标IP
2.数据链路层
  - 协议头:源端接路由器MAC+端接路由器MAC;每一条跳都修改协议头
1.物理层
  - 光/电/电磁波形式发射出去
  - 比特流
```
